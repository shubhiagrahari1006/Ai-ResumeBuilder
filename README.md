# AI Resume Builder

An AI-powered Resume Builder that helps users create professional resumes quickly and efficiently through an interactive web interface.

## Features

- Create professional resumes in minutes
- Clean and responsive user interface
- User-friendly form-based input
- Resume preview
- Easy customization
- Simple deployment

## Technologies Used

- Python
- HTML5
- CSS3
- JavaScript

```
│── LICENSE
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/shubhiagrahari1006/Ai-ResumeBuilder.git
```

2. Navigate to the project directory

```bash
cd Ai-ResumeBuilder
```

3. Install the required dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
